## How to Run App

1. cd to the repo
2. Install node modules

   - run `yarn`

3. cd to `src/server`
4. Install node modules

   - run `yarn`

5. Start the server

   - run `yarn start`

6. Run Build for either OS

   - for iOS
     - run `npx react-native run-ios`
   - for Android
     - run `npx react-native run-android`
